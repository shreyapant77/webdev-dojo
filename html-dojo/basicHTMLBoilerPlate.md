# this doc tells about the basic html boiler plate

```html
<!DOCTYPE html>
<html lang="en>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width,initial-scale=1.0">

    <title>this is shown on the tab</title>

</head>
<body>
    <h1>hello world</h1>

    <script src="script.js"></script>
```

# Student Marks Analyzer

## Task

Build an app that:

- add student marks into an array
- display all marks
- calculate total marks
- calculate average marks

Example:

Marks:

```js
[80, 90, 70]
```

Total:

```text
240
```

Average:

```text
80
```

---

## Summation Using Loops

Example:

```js
let sum = 0;

for(let i=0;i<arr.length;i++){
    sum += arr[i];
}
```

Result:

```text
sum contains total of all elements
```

---

## Average Formula

```js
average = total / numberOfElements
```

Example:

```js
240 / 3
```

Result:

```text
80
```

# 🥋 JavaScript Dojo — Project Based JavaScript Mastery

Welcome to the JavaScript Dojo section of Web Dev Dojo.

This roadmap is designed to help learners master JavaScript through hands-on mini projects instead of only watching tutorials.

The goal is to:

* build strong JavaScript foundations
* learn through practice
* understand frontend development deeply
* become ready for React and modern web development
* create real projects consistently

This is not:

```text
100 random projects
```

This is:

```text
100 structured projects designed to progressively teach JavaScript
```

---

# 📚 Learning Philosophy

Every project in this dojo follows this idea:

```text
Learn Concept → Think → Build → Debug → Improve → Commit
```

Each mini project contains:

```text
index.html
script.js
concepts.md
```

Where:

* `index.html` contains the webpage structure
* `script.js` contains the JavaScript logic
* `concepts.md` teaches the concepts required before building

The purpose is to encourage:

* active learning
* problem solving
* debugging skills
* independent thinking
